1. Introdução: O que é JSON?

O JSON (JavaScript Object Notation) é um formato leve de troca de informações entre sistemas do mundo inteiro. Apesar de ter "JavaScript" no nome, ele é como se fosse o "inglês" da programação: todas as linguagens de programação entendem JSON.

Sabe quando o aplicativo do Spotify pede dados para mostrar as músicas? O que viaja pela internet é um arquivo de texto no formato JSON.

2. Explicação Conceitual

Ele se organiza num sistema muito lógico chamado de Chave & Valor (Key & Value):

  • Chave (Key): A etiqueta que dá nome à informação (Ex: "Nome", "Idade").
  • Valor (Value): A informação real preenchida (Ex: "Guilherme", 28).

Dentro do JSON você pode ter informações em texto, números ou até em listas (uma prateleira de coisas dentro da ficha de alguém!).

4. Exemplos Práticos (Com Código Python)

No Python, o formato JSON é chamado de Dicionário (dict) quando está na memória. A lógica é exatamente a mesma. Vamos ver como criar um dicionário e, principalmente, como extrair apenas a informação que precisamos usando as Chaves.

cliente = {
"nome": "João Silva",
"cep": "01001-000",
"cidade": "São Paulo"
}

5. Desafios Práticos

Prepare-se para o simulador abaixo. Aqui estão seus objetivos lógicos:

  1. Criar um dicionário de produto: Faça uma variável chamada painel. Dentro de chaves {}, crie três dados para essa ficha técnica: "modelo", "potencia" e "preco". Preencha com os dados de um painel da loja SolarTech (ex: 500W).
  2. Acessar o modelo: Peça para o computador exibir (com o "print") apenas o modelo do painel. Lembre-se de colocar a chave correspondente entre colchetes (ex: painel["modelo"]).
  3. Bônus para o sistema: Faça um print exibindo também a "potencia" e o "preco" logo depois.
