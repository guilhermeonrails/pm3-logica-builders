1. Introdução: O que são Condicionais?

Você acorda, olha pela janela e pensa: "Se estiver chovendo, eu levo o guarda-chuva. Se não, eu vou de óculos de sol."

Na programação, os programas operam exatamente da mesma forma. A ferramenta que permite ao código tomar uma decisão se chama Estrutura Condicional ("If e Else").

É o momento em que o "robozinho" age escolhendo caminhos diferentes.

2. Explicação Conceitual (Verdadeiro ou Falso)

Para o computador tomar uma decisão, ele sempre avalia se:
uma afirmação é Verdadeira (True) ou Falsa (False)?

Exemplo: O meio de pagamento:

  • A afirmação "O método de pagamento é Pix" é Verdadeira?
  • Sendo Verdade (True) ⇒ Aplica 10% de desconto imediatamente.
  • Sendo Falso (False) ⇒ Segue cobrando o preço normal via Cartão de Crédito.

Basicamente, o sistema formula perguntas cuja resposta só pode ser sim ou não, e atrela uma ação diferente para cada resposta.

3. Verificando o estoque

Imagine que você tem um estoque de painéis solares.
Antes de vender, você precisa verificar a quantidade do pedido.

Regras:

  • Se o estoque for maior que 0, mostre "Compra aprovada!".
  • Se o estoque for 0, mostre "Sem estoque.".

Dica: crie duas variáveis, uma para o estoque e outra para a quantidade desejada.

estoque = 10
quantidade_desejada = 5
